## 1. Цель работы
Закрепить навыки работы с библиотекой **Pandas**: загрузка данных, фильтрация, вычисление статистик, проверка логических условий, а также построение простых визуализаций с помощью **Matplotlib/Seaborn**.

---

## 2. Исходные данные

### 2.1. Загрузка и просмотр
Данные загружаются из файла `adultdata.csv` в DataFrame `df`, после чего выводятся первые строки (`df.head()`), чтобы проверить структуру таблицы и корректность чтения.

### 2.2. Размерность и признаки
В датасете **32561 строка** и **15 столбцов**, среди которых присутствуют признаки `age`, `workclass`, `education`, `sex`, `hours-per-week`, `native-country` и целевая переменная `salary`.

---

## 3. Используемые инструменты
В работе применяются библиотеки: `numpy`, `pandas`, `matplotlib.pyplot`, `seaborn`; дополнительно увеличивается размер графиков через `plt.rcParams['figure.figsize'] = (18, 10)`.

---

## 4. Выполненные задания

### 4.1. Распределение по полу
Подсчёт выполнен через `df['sex'].value_counts()`.

**Результат:**
- `Male`: **21790**  
- `Female`: **10771**

**Вывод:** в выборке есть дисбаланс по полу — мужчин примерно в 2 раза больше, чем женщин. Это важно учитывать при сравнении статистик по полу.

---

### 4.2. Средний возраст мужчин
Средний возраст вычислен после фильтрации `sex == 'Male'` и применения `.mean()`.

**Результат:** средний возраст мужчин **≈ 39.43 года**.

**Вывод:** средний возраст мужской подвыборки соответствует взрослой трудоспособной аудитории.

---

### 4.3. Доля людей из Taiwan
Доля рассчитана как `mean() * 100` по булевой маске `native-country == 'Taiwan'`.

**Результат:** **0.16%**.

**Вывод:** наблюдений по Taiwan крайне мало, статистические выводы по этой стране делать сложно из-за малого объёма выборки.

---

### 4.4. Средний возраст и стандартное отклонение для группы `salary == '50K'`
После фильтрации по `salary == '50K'` вычислены `.mean()` и `.std()` для столбца `age`.

**Результаты:**
- Средний возраст: **≈ 38.58**  
- Стандартное отклонение: **≈ 13.64**

**Вывод:** в классе `50K` наблюдается заметный разброс по возрасту; внутри класса встречаются как молодые, так и значительно более взрослые респонденты.

---

### 4.5. Проверка утверждения про образование для `salary > 50K`
Проверено утверждение: «все люди с `salary > 50K` имеют образование только из набора (`Bachelors`, `HS-grad`, `11th`)» через `.isin(...)` и агрегирование.

**Результат:** утверждение **не подтверждается** (получено значение `False`).

**Вывод:** среди людей с доходом `> 50K` встречаются уровни образования, не входящие в указанный список.

---

### 4.6. Проверка условия: USA + 40 часов + `salary == '50K'`
Выполнена проверка комбинированного условия (`native-country == 'United-States'` AND `hours-per-week == 40` AND `salary == '50K'`) и выведен результат `value_counts()`.

**Результат:** `False 32561`.

**Вывод:** условие не выполнено для всех строк, то есть датасет целиком не описывается данной комбинацией признаков. Это комбинация описывает очень узкий фильтр.

---

### 4.7. Максимальное число рабочих часов и анализ распределения
Максимум `hours-per-week` найден через `.max()` и равен **99**.

Далее проверена корреляция с зарплатой для строк с максимальным числом часов.

**Результат:** получено распределение, где большинство наблюдений не удовлетворяют критерию (точное значение зависит от специфики условия в ноутбуке).

**Вывод:** люди с экстремально высоким числом рабочих часов не всегда имеют доход `> 50K`.

## 5. Визуализация данных

### 5.1. Столбчатая диаграмма распределения `education`
Строится график частот категорий `education` через `value_counts().plot(kind='bar')` с подписями осей и заголовком.

**Анализ:** график позволяет быстро определить наиболее частые уровни образования (обычно `HS-grad`, `Some-college`, `Bachelors`) и оценить дисбаланс между категориями.

**Вывод:** визуализация удобна для первичной оценки категориального признака, особенно при большом числе значений.

### 5.2. Гистограмма возраста и подсчёт плотных бинов
Строится гистограмма `age` с `bins=10`, затем считается число бинов, где частота **> 1600** через `np.sum(counts > 1600)`.

**Результат:** получено значение **6** (количество бинов с частотой > 1600).

**Вывод:** распределение возраста неравномерно — несколько интервалов (например, диапазоны 30–40 и 40–50 лет) имеют заметно большую концентрацию наблюдений, что типично для социально-демографических данных.
